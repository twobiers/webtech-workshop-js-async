Es ist Uhr

Diese Seite demonstriert, wie sich ein blockierender Event Loop äußert. Auf der linken Seite ist konzeptuell ein animierter Event Loop dargestellt, der mit einem Klick auf den Block Me button blockiert werden kann. Im Hintergrund wird eine komplizierte Berechnung vorgetäuscht. Das Ergebnis der Berechnung wird daraufhin in die darunter liegende Textbox geschrieben und die Webseite für 5 Sekunden blockiert.

Die Seite ist als Playground gedacht um mit dem Event Loop und die Auswirkungen von Blockierungen herumzuspielen.
